# CUDA 编程与高性能算子开发（课程体系）

> 覆盖从 CUDA 环境搭建到 FlashAttention 实现的完整 CUDA 学习路径，含 12 章基础 + 25 课时进阶。

---

## 1. 面向 AI-Infra 的 CUDA 编程与优化（12 章）

| 章节 | 内容 |
|------|------|
| **1. CUDA 环境搭建** | Linux/WSL 下 CUDA Toolkit 安装；Nsight Compute 使用；第一个向量加法 Kernel |
| **2. 线程模型与显存层次** | Thread/Block/Grid 层次；全局内存、共享内存、寄存器；Coalescing 优化；__syncthreads() |
| **3. Softmax 算子实现** | 数值稳定性问题；Naive → 共享内存优化 → Warp 级归约 → __shfl_sync |
| **4. Reduce 归约算子** | 树状归约与性能优化；warp 级+块级归约结合；大型数组分块归约 |
| **5. 矩阵乘法基础与分块** | Naive GEMM；2D 线程块设计；共享内存分块优化 |
| **6. 向量化访存与转置** | 向量加载指令；矩阵转置 CUDA 实现；非连续内存访问优化 |
| **7. 双缓冲流水线** | 流水线技术原理；双缓冲共享内存设计；计算与访存重叠 |
| **8. 原子操作与并发控制** | 原子操作类型与应用；性能优化策略；并发数据结构 |
| **9. 共享内存 Bank 冲突** | Bank 结构与冲突原理；数据重排与 Padding 解决策略 |
| **10. RMSNorm 算子实战** | RMSNorm vs LayerNorm；数值稳定实现；与 PyTorch 官方性能对比 |
| **11. FlashAttention CUDA 实现** | 注意力机制瓶颈；FlashAttention 核心优化思想；块级矩阵乘法+softmax；长序列支持 |
| **12. PyTorch 自定义 CUDA 算子** | C++/CUDA 扩展机制；Autograd 实现；Setup.py 配置；性能基准对比；项目集成 |

| 阶段 | 课时 | 内容 |

> 🔑 **核心要点**：CUDA 学习的关键路径是：理解线程模型 → 掌握内存层级 → 学会 Shared Memory Tiling → 用 Nsight 定位瓶颈。从 RMSNorm 到 FlashAttention，循序渐进，每个算子都是性能优化的练兵场。
