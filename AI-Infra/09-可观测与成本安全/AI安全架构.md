## 1. 容器安全

| 措施 | 说明 |
|------|------|
| **镜像漏洞扫描** | Trivy/Clair 扫描 CVE |
| **容器权限收紧** | 非 root 运行（runAsNonRoot） |
| **只读文件系统** | ReadOnlyRootFilesystem 防篡改 |
| **SecurityContext** | Privilege Escalation 禁用、Capabilities 最小化 |

---

## 2. 权限安全

- **最小权限原则**：RBAC 精细化授权
- **资源访问审计**：操作记录（Audit Log）
- **网络白名单**：NetworkPolicy 访问控制
- **零信任架构**：永远验证、永不信任

---

## 3. 数据安全

| 措施 | 说明 |
|------|------|
| **数据集脱敏** | PII 敏感信息保护 |
| **模型权重权限** | 访问控制、下载审计 |
| **传输加密** | TLS 1.3 / mTLS |
| **静态加密** | 存储层加密（AES-256） |

---

## 4. 集群安全

| 措施 | 说明 |
|------|------|
| **内网隔离** | VPC/VLAN 隔离 |
| **访问白名单** | IP/端口限制 |
| **漏洞修复基线** | 定期安全更新、CVE 修复 SLA |
| **操作审计** | 全操作记录（who/when/what） |
| **入侵检测** | Falco/Sysdig 运行时安全 |

## 5. AI 安全进阶

### 5.1 隐私计算

- **联邦学习（Federated Learning）**：数据不出本地，模型参数聚合
  - 横向联邦：同特征不同样本
  - 纵向联邦：同样本不同特征
- **多方安全计算（MPC）**：多个参与方联合计算，互不泄露原始数据
- **可信执行环境（TEE）**：Intel SGX/AMD SEV 硬件级隔离
- **差分隐私**：添加噪声保护个体数据，训练时梯度加噪

### 5.2 对抗防御

- 对抗样本检测：输入扰动识别与过滤
- 对抗训练：训练集中加入对抗样本增强鲁棒性
- 模型鲁棒性评估：攻击成功率/防御成功率指标

### 5.3 模型水印

- **白盒水印**：在模型权重/结构中嵌入标识
- **黑盒水印**：通过特定触发输入验证模型版权
- **用途**：模型版权保护、盗用溯源、合规审计

### 5.4 云边端安全协同

```
云端 → 全量数据加密 + 严格权限
边缘 → 数据预处理 + 过滤脱敏
端侧 → 隐私数据不出设备
```

---

> 🔑 **核心要点**：AI 安全需要从容器→权限→数据→集群四层纵深防御。隐私计算（联邦学习/TEE/差分隐私）和模型水印是 AI 特有的安全技术。安全不是一次性工作，而是持续的基线维护和审计。
