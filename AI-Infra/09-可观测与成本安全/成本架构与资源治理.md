## 1. 算力成本模型

### 1.1 GPU 小时单价

| 计费模式 | 价格（相对） | 适用场景 |
|----------|------------|----------|
| **按需** | 1x | 临时调试、紧急扩容 |
| **包年包月** | 0.5-0.7x | 长期稳定训练 |
| **抢占式（Spot）** | 0.2-0.3x | 可中断的离线训练 |

### 1.2 成本优化策略

- 抢占式实例 + 自动容错（Checkpoint 续训）
- 混合部署：核心推理按需 + 离线训练 Spot
- GPU 选型：不同任务选不同 GPU 型号（H100 训练、L40S 推理）

## 2. 资源利用率治理

### 2.1 核心手段

| 手段 | 说明 | 效果 |
|------|------|------|
| **碎片清理** | 资源碎片整理（小任务填碎片） | +5-10% 利用率 |
| **低负载整改** | 识别并回收低利用率任务 | +10-20% |
| **混部提效** | 训练 + 推理混合部署 | +15-30% |
| **错峰调度** | 利用低峰时段跑离线任务 | 摊平峰值 |

### 2.2 混部策略

```
白天（推理高峰）：GPU 优先推理
夜间（推理低谷）：GPU 切给离线训练
```

- 在线推理：不可抢占、独立 GPU
- 离线训练：可抢占、弹性 GPU

## 3. 存储成本优化

| 策略 | 具体措施 |
|------|---------|
| **冷热分层** | 自动迁移：热存储(SSD)→温存储(HDD)→冷存储(归档) |
| **压缩** | 模型权重压缩、数据集压缩 |
| **生命周期删除** | 过期 Checkpoint 自动清理（保留最近 N 个） |
| **冗余策略优化** | 副本数调整（3→2）、纠删码替代多副本 |

## 4. 成本可视化

### 4.1 多租户计费

- **部门成本分摊**：按 GPU·小时计费
- **预算管控**：配额管理、超额告警
- **资源账单**：精细化到用户/项目/模型维度

### 4.2 成本看板

```
部门A：本月 GPU 费用 ¥XX 万（占比 XX%）
部门B：本月 GPU 费用 ¥XX 万（占比 XX%）
存储费用：¥XX 万（热/温/冷层占比）
```

---

> 🔑 **核心要点**：成本优化是 AI Infra 最直接的价值体现。GPU 利用率从 50% 提升到 80%，意味着同样的硬件投入产出提升 60%。混部、错峰、Spot 实例、存储分层是四大核心降本手段。
