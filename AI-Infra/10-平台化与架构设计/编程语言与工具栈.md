# 编程语言与工具栈

> AI Infra 架构师需要掌握多语言 + 多中间件的技能组合。

---

## 1. 核心语言

| 语言 | 用途 | 深度要求 |
|------|------|----------|
| **Python** | AI 工程化、平台脚本、自动化 | 精通 |
| **Go** | 云原生组件、调度服务、中台 | 精通 |
| **C/C++** | 底层性能优化、GPU 原理 | 读懂+调优 |
| **CUDA** | 算子开发、通信库调优 | 基础 |
| **Shell/Rust** | 内核工具、高性能组件 | 辅助 |

### 语言应用场景

- **Python**：PyTorch 训练脚本、推理服务 API、数据处理 Pipeline、自动化运维
- **Go**：K8s Operator 开发、调度器插件、API 网关、中间件服务
- **C/C++**：NCCL 源码阅读、CUDA Kernel 开发、PyTorch C++ 扩展
- **CUDA**：自定义算子、通信优化、性能剖析

---

## 2. 中间件与工具

| 工具 | 用途 | AI Infra 场景 |
|------|------|--------------|
| **Prometheus/Grafana** | 监控可视化 | GPU 集群监控 Dashboard |
| **etcd** | 分布式配置 | 调度器配置中心、选主 |
| **Redis** | 缓存 | 推理请求缓存、任务队列 |
| **消息队列** | 异步通信 | 任务状态通知、日志收集 |
| **S3 SDK（MinIO）** | 对象存储 | 模型权重存储、数据集管理 |
| **Docker/containerd** | 容器工具链 | AI 镜像构建与运行 |
| **Helm** | K8s 包管理 | AI 组件一键部署 |

---

## 3. 工具链全景

```
开发：VS Code + Copilot + Claude Code
构建：Docker + Harbor + Helm
部署：K8s + Volcano + Kustomize
监控：Prometheus + Grafana + DCGM
日志：Fluentd + Loki + Grafana
追踪：OpenTelemetry + Jaeger
CI/CD：GitLab CI + ArgoCD
```

---

> 🔑 **核心要点**：Python + Go 是 AI Infra 的双语言主力，C++/CUDA 是进阶必学。工具链的选择遵循"社区成熟 + 可运维 + 与 K8s 生态集成"三原则。
