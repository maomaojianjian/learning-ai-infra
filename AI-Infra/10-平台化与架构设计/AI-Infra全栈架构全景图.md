## 2. 各层技术详解

### 2.1 AI 芯片（算）

| 技术 | 说明 |
|------|------|
| **NPU** | 神经网络处理单元。专为 AI 推理/训练设计的 ASIC 芯片，擅长矩阵运算。代表：华为昇腾、寒武纪。 |
| **GPU** | 图形处理单元。AI 训练的主力芯片。代表：NVIDIA A100/H100/B200、AMD MI 系列。 |
| **TPU** | 张量处理单元。Google 自研的 AI 专用芯片，针对 TensorFlow 优化。 |
| **CPU** | 中央处理器。在 AI 中主要用于数据预处理、控制流、小规模推理。 |

### 2.2 互联（网）+ 存储（存）

**互联：**
| 技术 | 说明 |
|------|------|
| **RDMA** | 远程直接内存访问，分布式训练的核心技术 |
| **IB** | InfiniBand，高性能计算网络标准 |
| **Ethernet** | 以太网，管理网络或非关键数据传输 |

**存储：**
| 技术 | 说明 |
|------|------|
| **HBM** | 高带宽内存，3D 堆叠 DRAM，与 GPU 封装在一起 |
| **存储阵列** | 分布式存储系统（Lustre/GPFS/MinIO） |

### 2.3 AI 集群

| 概念 | 说明 |
|------|------|
| **Super POD** | 由多个计算节点通过高速互联组成的大规模计算单元 |
| **K8s** | 资源调度、容器化部署、弹性伸缩的核心管理平台 |
| **Ubuntu 等 OS** | AI 服务器的主流操作系统 |

### 2.4 AI 训练框架

| 框架 | 说明 |
|------|------|
| **PyTorch** | 最主流的动态图训练框架，生态丰富 |
| **TensorFlow** | Google，工业界部署场景广泛应用 |
| **MindSpore** | 华为开源（昇思），原生支持自动并行 |
| **JAX** | Google 高性能 ML 库，基于 XLA |
| **Caffe** | Berkeley 视觉学习中心，早期 CV 主流，现被 PyTorch 取代 |

### 2.5 AI 推理框架

| 框架 | 说明 |
|------|------|
| **vLLM** | PagedAttention，高吞吐 LLM 推理 |
| **TensorRT** | NVIDIA 高性能推理优化 SDK |
| **Xinference** | 开源分布式推理平台 |
| **MNN** | Alibaba 轻量级推理引擎，移动端优化 |

### 2.6 加速库与三方框架

**训练加速：Megatron-LM、DeepSpeed、Colossal-AI、HuggingFace**
**应用框架：LlamaIndex、LangChain、AutoChain、RAGFlow、Milvus（向量数据库）**

### 2.7 模型算法与数据

| 类型 | 说明 |
|------|------|
| **MLM** | 多模态大模型（GPT-4V、Gemini、Qwen-VL） |
| **LLM** | 语言大模型（GPT-4、Claude、Llama、DeepSeek） |
| **VLM** | 视觉大模型（SAM、DINOv2） |
| **SoRA/ViT** | 视频生成模型 |
| **Stable Diffusion** | 文生图模型 |

### 2.8 模型应用与热点

RAG、具身智能、自动驾驶、生成式推荐
