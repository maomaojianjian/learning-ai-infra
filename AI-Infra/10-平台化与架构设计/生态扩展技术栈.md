## 1. 硬件互联新标准

### 1.1 CXL（Compute Express Link）

- **定义**：基于 PCIe 物理层的高速开放互联标准
- **三大协议**：
  - CXL.io：设备发现、配置（基于 PCIe）
  - CXL.cache：设备访问主机内存缓存
  - CXL.mem：主机访问设备内存
- **AI 场景价值**：内存池化突破 HBM 显存墙、CPU-GPU 零拷贝、异构统一内存空间

### 1.2 CPO（Co-Packaged Optics，共封装光学）

- **定义**：将光收发模块与交换芯片/GPU 封装在同一基板
- **价值**：带宽密度 100G+、功耗降低 50%、距离突破
- **进展**：NVIDIA Blackwell 已采用，TSMC COUPE 封装平台推进中

## 2. 数据处理单元 DPU

- **定义**：专为数据中心设计的可编程处理器，卸载 CPU 的网络/存储/安全任务
- **主流厂商**：NVIDIA BlueField-3（400Gbps）、Intel IPU、阿里云 CIPU
- **AI 应用**：RDMA/NCCL 通信卸载、存储加速（NVMe-oF/压缩/纠删码）、网络虚拟化卸载、硬件级多租户隔离

## 4. 存储与向量检索扩展

### 4.1 Ceph 分布式存储

- 统一架构：RBD（块）+ RGW（对象）+ CephFS（文件）
- CRUSH 算法：去中心化数据分布
- AI 场景：大容量数据集、Checkpoint 冷存储、多机房容灾

### 4.2 FAISS（向量相似度检索）

- Meta 开源，RAG 系统核心检索引擎
- 索引类型：IVF（倒排）、HNSW（图索引）、PQ（乘积量化）
- 支持十亿级向量 GPU 加速检索

### 4.3 零拷贝（Zero Copy）

- 消除 CPU 在内存区域间的数据复制
- 技术：mmap、sendfile、RDMA、GPU Direct RDMA
- AI 场景：训练数据加载零拷贝、推理请求零拷贝传输

## 8. 推理服务化进阶

### 8.1 Triton Inference Server（NVIDIA）

- 区别于 OpenAI Triton 语言，这是 NVIDIA 的推理服务框架
- 支持多框架（PyTorch/TF/ONNX/TensorRT）、多模型混部
- 动态批处理、模型并发执行、Pipeline 并行推理

### 8.2 PD 分离架构（Prefill-Decode Disaggregation）

- Prefill 节点：计算密集型，高显存带宽
- Decode 节点：显存密集型（KV Cache），低计算
- **优势**：独立扩缩容、资源利用率 50%→70%+、首 Token 延迟降低
- **实现**：SGLang Mooncake、vLLM disaggregated mode

## 9. 云原生进阶架构

- **Serverless**：按需扩缩容、按调用计费（Knative/AWS Lambda GPU版）
- **服务网格（Service Mesh）**：Sidecar 代理、mTLS、流量治理（Istio/Linkerd）
- **API 网关**：Token 级限流、多模型 API 格式统一、请求缓存去重（Kong/APISIX）

## 10. AI 安全进阶

- **隐私计算**：联邦学习、多方安全计算、TEE、差分隐私
- **对抗防御**：对抗样本检测、对抗训练、鲁棒性评估
- **模型水印**：白盒水印（权重嵌入）、黑盒水印（触发验证）

### 10+. 云边端协同

```
云端 (Cloud)          边缘 (Edge)          端侧 (Device)
大模型预训练     →    模型微调/推理    →    轻量化推理
千卡集群              边缘GPU服务器          手机/嵌入式
数据汇聚/标注         实时数据处理           离线推理
```

- **云端**：全量模型训练 + 模型仓库 + 数据管理
- **边缘**：模型适配微调 + 低延迟推理 + 数据预处理过滤
- **端侧**：量化轻量模型 + 离线可用 + 隐私数据不出设备
- **协同技术**：模型压缩分发、增量更新、联邦学习、知识蒸馏
