# 分阶段学习落地计划

> 从筑基到架构师专家的 5 阶段、15 个月完整学习路线。

---

## 阶段 1：筑基期（1-3 个月）

**目标**：吃透 Linux 底层 + GPU 硬件 + 基础分布式

| 时间 | 内容 |
|------|------|
| 上午 | 1 个内核知识点（内存/调度/网络/IO）+ 对应参数实操 |
| 下午 | GPU 硬件拓扑、NVLink/PCIe 基础、nvidia 工具实操 |
| 晚上 | 分布式基础理论 1 小节 + 复盘当日命令 |

**核心产出**：
- Linux 内核调优能力
- GPU 硬件拓扑理解
- K8s 深度使用
- NCCL/分布式训练基础

---

## 阶段 2：云原生 + AI 调度基建（3-6 个月）

**目标**：K8s 深度 + AI 容器 + Volcano 调度 + 异构设备管理

| 时间 | 内容 |
|------|------|
| 每日 | 拆解一个 K8s 组件源码/设计原理 |
| 实操 | device-plugin、runtime、调度策略调试 |
| 设计 | 多租户隔离、队列 QoS、资源配额 |

**核心产出**：
- K8s AI 生态改造能力
- Volcano 调度部署
- RDMA 网络配置
- 并行存储搭建

---

## 阶段 3：训推全链路（6-9 个月）

**目标**：分布式训练 + 大模型推理全栈

| 时间 | 内容 |
|------|------|
| 每周 | 吃透一种并行策略（DDP/FSDP/TP/PP） |
| 实操 | 部署 vLLM/Triton，压测、调优、量化落地 |
| 排查 | NCCL 通信调优、分布式故障排查 |

**核心产出**：
- 分布式训练并行原理
- 推理引擎部署优化
- 量化压缩落地

---

## 阶段 4：高阶架构（9-12 个月）

**目标**：存储分层 + 无损网络 + 稳定性 + 成本治理

| 时间 | 内容 |
|------|------|
| 架构 | 并行文件系统、对象存储架构设计 |
| 网络 | RoCE/IB 无损网络部署与问题排查 |
| 运营 | 监控全栈、告警体系、SLA、混部降本 |

**核心产出**：
- 存储分层架构设计
- 无损网络部署
- 全栈监控体系
- 成本优化方案

---

## 阶段 5：架构师专家能力（12-15 个月）

**目标**：自研平台设计 + 源码二次开发 + 全域架构评审

| 时间 | 内容 |
|------|------|
| 源码 | 拆解主流开源：NCCL、vLLM、containerd、Volcano |
| 设计 | 独立输出：中小规模 → 万卡集群演进架构方案 |
| 评审 | 方案写作、架构评审、容灾演练、故障复盘 |

**核心产出**：
- 开源项目二次开发能力
- 万卡集群架构设计
- 容灾安全体系

---

> 🔑 **核心要点**：学习路线遵循"基础 → 专项 → 全链路 → 架构"的渐进模式。每天的实践分为学（理论）和练（实操），每个阶段都有明确的能力验证产出。关键是保持节奏，不跳步。
